# Basic Form Validation App
This is a simple React app that demonstrates form validation using a custom hook called useInput. The app allows users to input data into a form and validates the entered values according to specific rules.
# Features
**Custom Hook**: The app uses a custom hook called useInput, which manages the form state and validation logic.

**Form Validation**: The entered values are validated based on provided validation rules.

**Error Handling**: If the entered value is invalid, the app displays an error message to the user.

**Form Reset**: Users can reset the form to its initial state.

**Dynamic Validation**: The validation rules can be customized and adapted to different input fields.

# Installation and Setup
1. Clone the repository to your local machine.    
2. Install the required dependencies using npm install.        
3. Start the development server using npm start.
4. Access the app through your browser at localhost:3000.
